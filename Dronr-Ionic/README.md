Dronr
=====================

An app for detecting weather hazards and no fly zones for UAVs

## Using this project

Ensure Ionic is installed

```bash
$ npm install -g cordova ionic
```

Then run: 

```bash
$ ionic build
```

To emulate on iOS:

'''bash
$ ionic emulate ios --target="iPhone-6s"
'''

To emulate in a web server:

'''bash
$ ionic serve
'''
